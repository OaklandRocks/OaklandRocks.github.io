An overview of merge sort quick sort and bubble sort

Merge sort

Merge sort is a divide and conquor sorting algorithim that takes half of the array and sorts it, and takes the other half and sorts that. After its done sortng both sides it merges it together. SO if we were to pseudocode it would look somthing like this

    If array has 1 element
    return Array # it's all ready sorted
    else
    split array into Left Half and Right Half
    SortedLeftHalf = MergeSort(LeftHalf)
    SortedRightHalf = MergeSort(RightHalf)
    Sorted Array = Merge(SortedLeftHalf, SortedRightHalf
    return SortedArray

We need to come up with a function/method to merge the array

    Create method Merge(firstArray, secondArray)
    set OutputArray to empty
    While neither the firstArray nor secondArray are empty
    compare the starting elements of firstArray and secondArray
    take the smaller one off and add it to the end of the OutputArray
    unless both firstArray and SecondArray are empty
    let remainder be the non-empty one
    add remainder to the end of the OutputArray
    return OutputArray

So we run a loop on the firstArray and conitue untill it's empty then dump the sorted contents to the OutputArray

Quick sort.

Quick sort is a O(n log n) algorithm. So what does that mean? Baiscally all you really need to know is that it is a divide and conquer algorithm. It first divides a large arry into smaller sub-arrys. Then it recursivly sorts the sub-arrays. from the wiki page here is the pseudocode for quicksort

To better understand this we should look at it in Ruby code

      def quicksort(array, from=0, to=nil)
      if to == nil
      # Sort the whole array, by default
      to = array.count - 1
      end
      if from >= to
      # Done sorting
      return array
      end
      # Take a pivot value, at the far left
      pivot = array[from]
      # Min and Max pointers
      min = from
      max = to
      # Current free slot
      free = min
      while min < max
      if free == min # Evaluate array[max]
      if array[max] <= pivot # Smaller than pivot, must move
      array[free] = array[max]
      min += 1
      free = max
      else
      max -= 1
      end
      elsif free == max # Evaluate array[min]
      if array[min] >= pivot # Bigger than pivot, must move
      array[free] = array[min]
      max -= 1
      free = min
      else
      min += 1
      end
      else
      raise "Inconsistent state"
      end
      end
      array[free] = pivot
      quicksort array, from, free - 1
      quicksort array, free + 1, to
      end
