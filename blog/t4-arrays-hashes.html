Arrays and Hashes explained

Arrays

This is how you create an array

  
ary = [1, "two", 3.0] #=> [1, "two", 3.0]

An array can also be created by explicitly calling ::new with zero, one (the initial size of the Array) or two arguments (the initial size and a default object).

  
ary = Array.new    #=> []
Array.new(3)       #=> [nil, nil, nil]
Array.new(3, true) #=> [true, true, true]

A way to access objects within an array. you can start from the end of the array using negative numbers from -1 and you can even grab ranges of values at a time

  

 arr = [1, 3, 5, 7, 2]   # favorite way to declare an array
=> [1, 3 ,5 ,7 ,2]
> arr[0]
=> 1
> arr[-1]
=> 2
> arr[1..3]
=> [3, 5, 7]          # this returned an array!
> arr.slice(1..3)
=> [3, 5, 7]          # same as using [1..3]
> arr[1..200000]      
=> [3, 5, 7, 2]       # no error... silently cuts off at the end

An array is a great way to store whatever you need here are few useful methods that you can use on an array

  

#max to find the biggest value of an array

#min to find the smallest value of an array

#uniq to remove all duplicates from your array

#size to find out how big the array is

#shuffle will mess up your whole array by putting it in random order

#sort will clean it up again for you by putting your array in order. 

#sample picks out a totally random value from the array... good for gambling games!

#first gives you the first item (but doesn't remove it, so it's same as [0]) but can be more descriptive of your code's intent.

#last is same as [-1]

Hashes

According the the ruby docs a has is

A Hash is a dictionary-like collection of unique keys and their values. Also called associative arrays, they are similar to Arrays, but where an Array uses integers as its index, a Hash allows you to use any object type. Hashes enumerate their values in the order that the corresponding keys were inserted. A Hash can be easily created by using its implicit form:
By looking into a hash we can define the different parts of it

  
grades = { "Jane Doe" => 10, "Jim Doe" => 6 }

  • "Jane Doe" is the key and "10" is the value.
