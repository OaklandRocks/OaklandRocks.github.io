JS vs Ruby

How to make an array

    JS

  • var array = [1,"hello", true];
  • array.push("3") #=> 1,"hello", true, "3"
  • array.slice(1,3) #=> "hello", true, "3"
  • array[0] #=> [1]
  • array.indexOf("hello") #=> 1
    •

    RUBY

  • array = [1,"hello", true]
  • array.push("3") #=> 1,"hello", true, "3"
  • array.slice(1,3) #=> "hello", true, "3"
  • array[0] #=> [1]
  • array.index("hello") #=> 1
    •

    When defining arrays in Javascript you should remember to use "var" when assinging an array to the vairable, as for ruby you do not need to use the word "var".
